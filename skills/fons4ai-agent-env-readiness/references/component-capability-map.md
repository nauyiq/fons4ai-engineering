# 组件能力映射参考

本文件用于在执行 `fons4ai-agent-env-readiness` 时按需读取，帮助判断不同外部组件能为 Agent 验证提供什么能力。

## MySQL / 关系型数据库

- 验证价值：表结构、字段语义、状态流转、数据修复影响、幂等记录、任务执行结果。
- 推荐能力：只读查询、schema 查看、测试环境可控写入、查询结果脱敏。
- 可选工具：数据库 MCP、项目 DAO 测试、SQL CLI、只读账号、测试容器。
- 典型证据：SQL 摘要、影响行数、关键状态字段、测试用例结果。
- 安全边界：生产默认只读；禁止输出连接串、账号、手机号、身份证、银行卡等敏感值。

## Redis / 缓存

- 验证价值：缓存命中、分布式锁、幂等 key、限流计数、临时状态、延迟队列辅助状态。
- 推荐能力：只读 key 查询、TTL 查询、类型检查、测试环境受控删除或写入。
- 可选工具：Redis MCP、redis-cli、项目缓存封装测试、本地 Redis。
- 典型证据：key 模式、TTL、数据类型、脱敏后的 value 摘要。
- 安全边界：禁止生产环境批量扫描、批量删除或写入；避免输出完整业务敏感缓存值。

## RocketMQ / 消息系统

- 验证价值：消息是否投递、消费者是否在线、消费位点、重试/DLQ、异步流程闭环。
- 推荐能力：topic/consumer group 查询、消息轨迹、测试环境发送消息、消费日志关联。
- 可选工具：MQ MCP、管理控制台、mqadmin、项目集成测试、日志平台。
- 典型证据：topic、consumer group、message key、消费状态、重试次数、相关日志。
- 安全边界：生产环境默认只查不发；禁止随意重投、跳过或修改消费位点。

## Nacos / 配置中心

- 验证价值：配置是否存在、命名空间/分组是否正确、动态配置是否生效、服务发现状态。
- 推荐能力：配置只读查看、历史版本查看、测试环境配置发布验证、服务实例查询。
- 可选工具：Nacos MCP、控制台、OpenAPI、项目配置加载日志、本地配置替代。
- 典型证据：namespace、group、dataId、配置摘要、版本号、加载日志。
- 安全边界：禁止输出密钥配置值；测试环境以外不自动发布配置。

## 日志 / 链路追踪

- 验证价值：接口调用、异常堆栈、异步链路、外部接口返回、任务执行时间线。
- 推荐能力：按 traceId、订单号、用户脱敏标识、message key 查询日志。
- 可选工具：日志平台 MCP、ELK/OpenSearch、应用日志文件、APM。
- 典型证据：时间范围、traceId、关键日志摘要、异常类型。
- 安全边界：日志内容必须脱敏，不输出 token、密钥、完整用户隐私。

## 测试环境 / 本地替代

- 验证价值：在不触碰生产的前提下形成端到端或近端到端闭环。
- 推荐能力：docker compose、Testcontainers、本地 mock、沙箱账号、测试数据回滚。
- 可选工具：项目脚本、CI 任务、本地依赖容器、接口自动化集合。
- 典型证据：启动命令、测试命令、接口请求摘要、断言结果。
- 安全边界：测试数据必须可回收；不把本地 mock 结果冒充真实环境验证。
